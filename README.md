# 🎬 Movie Recommender System

A **content-based Movie Recommender System** built using **Python and Streamlit** that suggests similar movies based on user selection. The system uses **cosine similarity** to find related movies and fetches posters using the **TMDB API**.

---

## 🚀 Features

* Recommend top 5 similar movies
* Interactive UI using Streamlit
* Fetch movie posters dynamically
* Fast and simple recommendation logic

---

## 🛠️ Tech Stack

* Python
* Pandas
* Scikit-learn
* Streamlit
* TMDB API

## ▶️ How to Run

1. Clone the repository:

```
git clone https://github.com/your-username/movie-recommender-system.git
```

2. Navigate to project folder:

```
cd movie-recommender-system
```

3. Install dependencies:

```
pip install -r requirements.txt
```

4. Run the app:

```
streamlit run app.py
```
